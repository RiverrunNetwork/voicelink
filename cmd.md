# 目前支持的指令
```java
查看大图| 商品大图
{
	"action" : "watch",
	"filter" : "bigPic"
}


查看详情 | 图文详情
{
	"action" : "watch",
	"filter" : "detail"
}

宝贝评价 | 查看评价 | 商品评价 |评价
{
	"action" : "watch",
	"filter" : "evaluate"
}

店铺首页 | 查看店铺首页 | 店铺页 
{
	"action" : "watch",
	"filter" : "homePage"
}

宝贝推荐 | 看看宝贝推荐 
{
	"action" : "watch",
	"filter" : "recommend"
}


上拉
{
	"action" : "move",
	"moveDir" : "up"
}


下拉
{
	"action" : "move",
	"moveDir" : "down"

	"action" : "sureBuy"
}
```
